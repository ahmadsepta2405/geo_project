# Langkah pembuatan project
1. memastikan merubah dependency di pubspec.yaml, mengganti beberapa dependency yaitu gelocotaor, geocoding, google_maps_flutter.
2. setelah itu jalankan perintah flutter pub get di terminal.
3. menambahkan permission pada file AndroidManifest.xml
   <img width="1580" height="444" alt="image" src="https://github.com/user-attachments/assets/3ef92fc4-c4b3-4665-923b-f02f575e0a97" />
4. menambahkan meta data API key di dalam aplication karena menggunakan google maps
   <img width="1212" height="482" alt="image" src="https://github.com/user-attachments/assets/98660733-7af2-482e-9338-4c70e99f8237" />
5. update main.dart menambahkan tombol mengambil lokasi, menampilkan koordinat
6. jalankan projectnya menggunakan "flutter run"
